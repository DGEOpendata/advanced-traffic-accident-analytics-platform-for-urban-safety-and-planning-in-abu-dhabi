# Advanced Traffic Accident Analytics Platform

This project leverages the "Traffic Accident Reports in Abu Dhabi Dataset 2023" to create an analytics platform aimed at improving road safety, urban planning, and public awareness. The platform offers visualizations, predictive models, and download options to cater to various stakeholders.

## Features
- **Data Visualization**: Visualize accident trends by month, location, and weather conditions.
- **Predictive Modeling**: Predict accident frequency based on historical trends.
- **User-Friendly Exports**: Export data and insights in accessible formats (XLSX, CSV).

## Requirements
- Python 3.8+
- Pandas
- Matplotlib
- Scikit-learn
- Seaborn

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/user/traffic-analytics-platform.git
   
2. Navigate to the project directory:
   bash
   cd traffic-analytics-platform
   
3. Install dependencies:
   bash
   pip install -r requirements.txt
   

## Usage
1. Place the dataset file (`Traffic_Accident-v3.0_AbuDhabi_2023.csv`) in the project directory.
2. Run the analytics script:
   bash
   python analytics.py
   
3. Explore the generated visualizations and predictions.
